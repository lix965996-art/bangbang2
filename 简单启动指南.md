# 智慧农场管理系统 - 简单启动指南

## 启动前先配置环境变量

当前项目不再内置任何 API Key，启动前需要先配置环境变量。

---

## 🚀 最简单的启动方式

### 方法一：命令行启动
```bash
# 进入springboot目录
cd springboot

# 直接运行
java -jar target/springboot-0.0.1-SNAPSHOT.jar
```

### 方法二：IDE 启动
1. 先确认环境变量已经配置完成
2. 完全关闭并重新打开 IDEA
3. 直接运行后端启动类

---

## 🔧 应用说明

### 环境变量配置
启动前请至少配置以下变量：
- **QWEN_API_KEY**: 通义千问 API 密钥
- **AMAP_JS_KEY** 或 **AMAP_WEB_KEY**: 至少配置一个高德地图密钥
- **AMAP_JS_SECURITY_KEY**: 前端地图页面使用的安全码

### 启动成功标志
当你看到以下日志时，说明启动成功：
```
:: Spring Boot ::                (v2.5.9)
Tomcat initialized with port(s): 9090 (http)
Starting service [Tomcat]
Servlet engine: [Apache Tomcat/9.0.56]
Root WebApplicationContext: initialization completed
```

---

## 🌐 访问地址

- **主页**: http://localhost:9090
- **API文档**: http://localhost:9090/swagger-ui.html
- **健康检查**: http://localhost:9090/actuator/health

## 🔍 功能测试

### 1. AI对话测试
访问 http://localhost:9090/api/chat/ask
发送请求：
```json
{
  "question": "今天天气怎么样？"
}
```

### 2. 天气API测试
访问 http://localhost:9090/aether/weather/now

---

## 📝 注意事项

1. **首次启动**：应用可能需要几秒钟初始化数据库连接
2. **端口占用**：如果9090端口被占用，会启动失败
3. **浏览器访问**：建议使用Chrome或Edge浏览器

---

## 🔧 如果遇到问题

1. **端口被占用**
   ```bash
   # 查看端口占用
   netstat -ano | findstr :9090

   # 杀死进程
   taskkill /F /PID <进程ID>
   ```

2. **启动失败**
   - 检查Java是否已安装
   - 确保在springboot目录下运行
   - 查看控制台错误信息
