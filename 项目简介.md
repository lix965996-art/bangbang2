# 项目简介

## 这是什么项目

这是一个以“智慧农场”为主题的全栈管理系统，目标是把农场经营、环境监测、地图展示、物资流转和 AI 能力放到同一个平台中统一管理。

项目包含两个主要部分：

- `springboot/`：后端服务，负责数据接口、权限、业务逻辑、外部服务调用
- `vue/`：前端页面，负责管理后台、数据可视化、地图和交互展示

## 它能做什么

项目目前覆盖的主要能力包括：

- 农田信息管理
- 员工与角色权限管理
- 公告管理
- 采购、库存、销售管理
- 环境监测与可视化大屏
- 高德地图农场地理展示
- 农场 3D 展示
- 果蔬识别与检测分析
- 基于通义千问的智能问答与辅助分析

## 适合什么用途

- 毕业设计
- 课程设计
- 项目答辩展示
- 个人作品集
- 农业数字化系统原型演示

## 当前需要记住的环境变量

后端：

- `QWEN_API_KEY`
- `AMAP_JS_KEY`
- `AMAP_WEB_KEY`
- `AMAP_JS_SECURITY_KEY`

前端：

- `VUE_APP_AMAP_JS_KEY`
- `VUE_APP_AMAP_SECURITY_CODE`

说明：

- `VUE_APP_AMAP_JS_KEY` 应与 `AMAP_JS_KEY` 使用同一个值
- `VUE_APP_AMAP_SECURITY_CODE` 应与 `AMAP_JS_SECURITY_KEY` 使用同一个值

## 启动方式

后端：

```bash
cd springboot
mvn spring-boot:run
```

前端：

```bash
cd vue
npm install
npm run serve
```
